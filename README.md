Задание

На странице, которая тянется 990-, выводится 6+ новостных сообщений
Каждое сообщение состоит из:
- заголовка (всегда однострочный),
- даты (в формате ЧЧ мес ГГГГ),
- изображения (всегда тянется на 100% по ширине);
- анонса.

Необходимо сверстать так, чтобы сообщения имели равную ширину и тянулись, а отступы между сообщениями были фиксированными, по 12px. Обратите внимание на адаптивность, на малых разрешениях количество сообщений на строке постепенно уменьшается. Брейкпойнты определить самостоятельно.
!В IE8 отступ между сообщениями пусть тянется, в %.

Используются шрифты PT Sans Narrow и PT Sans Narrow Bold.
PSD нет :)

Требования:
 - PixelPerfect;
 — HTML 5;
 — Кроссбраузерность (FF, IE8+, Opera 12+, Chrome);
 — Использование принципа изящной деградации;
 — Аккуратный код.

 Большим плюсом будет использование Sass/Scss.
